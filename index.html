Variables y Tipos de Datos en JavaScript

Introducción a Variables

Las variables son contenedores para almacenar datos. En JavaScript, puedes crear variables usando las palabras clave var, let y const.

  1. var: Es la forma más antigua de declarar variables y tiene un alcance funcional. Se recomienda evitar su uso en código moderno.
  2. let: Se utiliza para declarar variables que pueden cambiar su valor y tiene un alcance de bloque.
  3. const: Se utiliza para declarar constantes, es decir, valores que no cambiarán.

Tipos de Datos en JavaScript

Los tipos de datos en JavaScript son categorías que definen el tipo de valor que una variable puede contener y cómo se puede manipular. Estos tipos de datos se dividen en dos grandes grupos: primitivos y no primitivos.

Tipos Primitivos: Son los tipos de datos básicos.

  1. string: Cadena de texto.
    Ejemplo: "Hola"
  2. number: Números
    Ejemplo: 42
  3. boolean: Verdadero o falso
    Ejemplo: true o false
  4. null: Ausencia intencionada de valor.
  5. undefined: Valor no definido.
  6. symbol: Tipo de dato único e inmutable.
    Usado en situaciones avanzadas
  7. Objeto: Colección de datos y más compleja.
    Ejemplo: { nombre: "Juan", edad: 30 }
